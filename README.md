# 图片表格识别工具

基于 AI 大模型的图片表格识别工具，支持手写识别、自动学习修正、内置数据计算函数。

## 功能特性

- **AI 视觉识别** — 调用通义千问大模型，手写/打印均可识别
- **自动纠偏** — 图片歪了自动拉正
- **实时旋转** — 拖动滑块实时预览旋转效果
- **自动学习修正** — 修正识别错误后点击"记住修正"，下次自动应用
- **内置计算函数** — 平均数、求和、最大值、最小值、中位数、标准差、方差、计数
- **最小二乘法拟合** — 支持列对列、行对行、列对行三种模式
- **导出 Excel** — 识别结果一键导出为 xlsx 文件
- **历史记录** — 保存识别记录，随时查看和导出
- **手机适配** — 支持手机拍照上传，响应式界面

## 截图

> 上传图片 → 旋转调整 → AI 识别 → 编辑修正 → 计算分析 → 导出 Excel

## 快速开始

### 1. 安装依赖

```bash
pip install streamlit openai pandas numpy opencv-python Pillow rapidocr-onnxruntime
```

### 2. 获取 API 密钥

前往 [阿里云 DashScope](https://dashscope.console.aliyun.com/) 注册，获取 API Key（有免费额度）。

### 3. 运行

```bash
streamlit run app.py
```

浏览器自动打开 `http://localhost:8501`，在左侧设置中输入 API Key 即可使用。

## 使用方法

1. **上传图片** — 点击上传或拖拽图片到页面
2. **调整角度** — 使用旋转按钮或滑块调整图片角度
3. **开始识别** — 点击"开始识别"按钮
4. **编辑修正** — 直接点击单元格修改识别错误的内容
5. **记住修正** — 点击"记住修正"，系统会学习你的修正
6. **数据计算** — 选择函数和列/行进行计算
7. **导出** — 下载 Excel 文件或保存到历史记录

## 内置计算函数

| 函数 | 说明 |
|------|------|
| 平均数 | 求选中数据的算术平均值 |
| 求和 | 求选中数据的总和 |
| 最大值 | 求选中数据的最大值 |
| 最小值 | 求选中数据的最小值 |
| 中位数 | 求选中数据的中位数 |
| 标准差 | 求选中数据的样本标准差 |
| 方差 | 求选中数据的样本方差 |
| 计数 | 统计非空数据个数 |
| 最小二乘法 | 线性拟合 y = ax + b |

## 技术栈

- **前端**: Streamlit + HTML/JavaScript
- **AI 识别**: 通义千问视觉大模型 (qwen-vl-plus / qwen-vl-max)
- **本地 OCR**: RapidOCR (备用)
- **图像处理**: OpenCV + Pillow
- **数据处理**: Pandas + NumPy
